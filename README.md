## üß† Strategic Logic (The "Contracts")

### 1. The Generation Freeze
Once a session is "Forged," all setup parameters are locked. Any change to the map, epochs, or player count forces a full recalculation to maintain project integrity.

### 2. Eligibility vs Weighting
The Forge distinguishes between **Hard Rules** (Exclusions) and **Soft Suggestions** (Weights).
- **Hard Rule**: If the match ends in the Bronze Age, Nanotech is *strictly* excluded.
- **Soft Suggestion**: If the Map is "Small Islands," Naval technologies receive a high priority weight.

### 3. Cost Calculation Formula
We use the exact in-game logic for multiple bonus costs:
`final_cost = base_cost + (taken_count_in_heading √ó multiple_bonus_cost)`
